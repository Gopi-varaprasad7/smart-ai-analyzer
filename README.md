# Smart AI Analyzer & Predictor 🚀

## 📌 Project Overview

This project is part of my AI Mastery journey.

It analyzes student data and predicts final scores using Machine Learning.

## 🧠 Features

- Data loading & cleaning
- Data visualization
- Feature analysis
- Model training (Linear Regression)
- Prediction system

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Dataset

Custom student dataset with:

- hours_studied
- attendance
- previous_score
- sleep_hours

## 🎯 Goal

To understand end-to-end ML pipeline deeply.

## 🚀 How to Run

```bash
pip install -r requirements.txt
python main.py
```
